# Helm: Basic Cirros deployment

Requirements:
- cirros image loaded into 'cirros-image' pvc

## Helm deployment

```
$ helm install my-cirros ./cirros-basic -n default
NAME: my-cirros
LAST DEPLOYED: Fri Oct  6 06:16:09 2023
NAMESPACE: default
STATUS: deployed
REVISION: 1
TEST SUITE: None
```

## OC commands

```
$ oc get vm my-cirros
NAME        AGE   STATUS    READY
my-cirros   64s   Running   True

$ oc get vmi my-cirros
NAME        AGE   PHASE     IP            NODENAME   READY
my-cirros   68s   Running   10.128.1.26   ocp-bm1    True

$ oc get service my-cirros-ssh
NAME            TYPE       CLUSTER-IP       EXTERNAL-IP   PORT(S)        AGE
my-cirros-ssh   NodePort   172.30.187.207   <none>        22:30932/TCP   95s
```
