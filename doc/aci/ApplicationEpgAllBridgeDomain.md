# Application Endpoint Group (EPG)

## Get EPGs' Bridge Domain properties

Use '--view bd' to get bridge domain properties of selected epgs
- epg name, application profile and tenant
- bridge domain name
- bridge domain subnets with usage
- associated L3 Out
