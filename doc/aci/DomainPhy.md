# Phy Domain

Get default properties of [all](./DomainPhyAll.md) Phy Domains in selected APIC.

Filter options:
  - [name](./DomainPhyName.md)

Output options:
  - [default](./DomainPhyAll.md)
  - [json](./DomainPhyOutputJson.md)

Command options

```
# iserver get aci domain phy --help

Usage: iserver.py get aci domain phy [OPTIONS]

  Get aci domain phy

Options:
  --apic TEXT                  APIC name
  --ip TEXT                    APIC IP
  --username TEXT              APIC Username
  --password TEXT              APIC Password
  --name TEXT                  Filter by pool name
  -o, --output [default|json]  [default: default]
  --no-cache                   Disable cache
  --devel                      Developer output
  --help                       Show this message and exit.

Info: finished in 52 ms and logs saved in /tmp/iserver\0796c42357c5
```
