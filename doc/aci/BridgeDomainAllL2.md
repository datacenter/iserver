# Bridge Domain

## Get L2 properties of bridge domains

Use '--view l2' to get L2 forwarding related properties of selected bridge domains
- L2 unknown unicast
- ARP flooding
- multi destination flooding
- clear remote MAC entries
- scaled L2 only (legacy) mode
